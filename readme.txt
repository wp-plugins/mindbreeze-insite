Replace the standard WordPress search with your Mindbreeze InSite search.

== Description ==

The Mindbreeze InSite plugin replaces the standard WordPress search by linking to your Mindbreeze InSite account.

== Installation ==

Automatic Installation:
1. In your WordPress administration, go to the Plugins page and click on the Add New link.
2. Search for the Mindbreeze InSite Plugin and click Install Now and activate this plugin. You will now have a new Mindbreeze InSite option under Settings.
3. Go to the new Mindbreeze InSite Settings page and enter the configuration Snippet from the Mindbreeze InSite Demo Page.

Manual Installation:
1. Download the plugin.
2. Unzip the mindbreeze.zip file.
3. Upload the mindbreeze directory to your `wp-content/plugins/` directory.
4. In your WordPress administration, go to the Plugins page.
5. Activate the Mindbreeze InSite plugin. You will now have a new Mindbreeze InSite! option under Settings.
6. Go to the new Mindbreeze InSite Settings page and enter the configuration Snippet from the Mindbreeze InSite Demo Page.
